# データフィードジョブの再実行

Adobe Analytics の データフィード UI を使用すると、データフィードの管理がはるかに容易になります。データフィードを管理するアクセス権を持っている場合は、カスタマーサポートチームに最近のデータフィードファイルを再送してもらわなくても、データフィード UI からデータフィードジョブを再実行できます。
