# データフィードジョブの再実行

Adobe Analyticsのデータフィード UI を使用すると、データフィードの管理がはるかに容易になります。 データフィードを管理するアクセス権を持っている場合は、カスタマーサポートチームにデータフィードファイルを再送してもらわなくても、データフィード UI でデータフィードジョブを再実行できます。
