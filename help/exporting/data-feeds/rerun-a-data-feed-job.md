# データフィードジョブの再実行

Adobe AnalyticsのデータフィードUIを使用すると、データフィードの管理がはるかに容易になります。 データフィードの管理にアクセスできる場合は、カスタマーサポートチームに最近のデータフィードファイルを再送信してもらうことなく、データフィードUIを使用してデータフィードジョブを再実行できます。
