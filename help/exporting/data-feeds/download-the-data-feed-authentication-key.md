# データフィード認証キーのダウンロード

データフィードを作成すると、生データファイルの送信先と各ファイルに含めたい内容をアドビが把握できます。Analyticsでは、データフィードの SFTP サポートを利用できます。SFTP ホスト、ユーザー名、および宛先サイトに有効な RSA または DSA 公開キーが含まれている必要があります。フィードの作成時に、適切な公開キーをダウンロードできます。
