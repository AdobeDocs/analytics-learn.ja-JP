# [!UICONTROL セグメント化のディメンションモデル]

 ディメンションモデルは、任意のディメンションセグメントの歯車を通じ  て使用できます。これらのモデルは主に順次セグメント化に役立ち、[!UICONTROL ディメンション]（デフォルト）、インスタンス、または[!UICONTROL ディメンション]の繰り返されないインスタンスの永続値をセグメント化できます。
