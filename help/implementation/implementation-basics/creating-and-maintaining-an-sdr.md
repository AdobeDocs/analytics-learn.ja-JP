# Solution Design Reference(SDR)ドキュメントの作成と保守

Solution Design Reference(SDR)ドキュメントは、基本的にはサイトの設計図です。 このビデオでは、ビジネス要件ドキュメントとの組み合わせでビジネス要件を作成する方法を示します。Adobe Analyticsを実装する際に、また将来参照する際に、いつでも真実の情報を得ることができます。
