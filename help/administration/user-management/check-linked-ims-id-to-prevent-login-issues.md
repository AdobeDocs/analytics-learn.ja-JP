# ログインの問題を防ぐために、「リンクされたIMS ID 」を確認します。

Adobe Analyticsで誤った、またはリンクされたIMS IDが見つからないと、ユーザーのログインに問題が発生する可能性があります。 ユーザーは、リンクされたIMS ID自体を確認し、カスタマーサポートに問い合わせて問題を報告できるので、一般的なトラブルシューティング手順を実行する手間を省くことができます。
