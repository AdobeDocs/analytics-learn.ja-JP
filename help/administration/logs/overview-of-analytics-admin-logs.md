# Analytics 管理ログの概要

ユーザーのログイン時間、使用状況、アクセス、レポートスイート、管理者による変更などを確認するのに役立つログファイルです。管理ログは、管理ツールで管理者が実行した変更をすべて報告します。ログは、3 つのログのいずれかからユーザー定義のレポートへのゲートウェイを提供します。
