# フロービジュアライゼーションの作成

フロービジュアライゼーションは、顧客がブランドでたどる正確なジャーニーを調査するために使用します。フローは、ページ、マーケティングチャネル、および収集した他のディメンションやディメンションの組み合わせに適用できます。
