# Analysis Workspace での埋め込みコンポーネントの共有

Analysis Workspace のプロジェクトを共有する際に、プロジェクトに含めたセグメントや指標を同僚と共有することを忘れてしまうことがあります。このビデオでは、プロジェクトを共有するプロセスでセグメントと指標を共有し、意図した内容が毎回同僚に表示されるようにする方法を説明します。
