# Analysis Workspace でのプロジェクト共有

プロジェクトの共有は、Analysis Workspace から組織のユーザーに至るデータや洞察を民主化する 1 つの方法です。受信者を、使用するプロジェクトエクスペリエンスに応じて、編集、複製、表示の 3 つのプロジェクトの役割のいずれかに受信者を配置できます。
