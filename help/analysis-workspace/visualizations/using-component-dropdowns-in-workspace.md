# ワークスペースでのコンポーネントドロップダウンの使用

コンポーネントのドロップダウンを使用すると、互換性のあるコンポーネントのリストから選択して分析を作成できます。 必要なものが見つからない場合は、ドロップダウンから直接新しいコンポーネントを作成することもできます。 これらのスペースは、より高度なユースケースの場合には、ドロップゾーンとしても引き続き機能します。
