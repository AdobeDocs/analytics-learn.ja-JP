# ビジュアライゼーションの精度の設定

トレンドビジュアライゼーション設定で精度セレクターを使用すると、データの表示方法を非常に簡単に調整できます。

>[!VIDEO](https://video.tv.adobe.com/v/23548/?quality=12&learn=on)

精度設定により、グラフとソーステーブルの両方が調整され、テーブルの非表示を解除および操作する手順が保存されます。
