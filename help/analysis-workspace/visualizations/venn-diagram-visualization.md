# Analysis Workspace のベン図ビジュアライゼーション

ベン図ビジュアライゼーションを使用して、Analysis Workspace でセグメントの重複と積集合を視覚的に表示できます。多数のインサイトを得ることができるベン図を数秒で簡単に作成できます。このビデオでは、その方法を説明します。
